ANTI-PATTERNS CATALOG

1. GOD FUNCTION
   - Single function >100 lines doing multiple things
   - Fix: Extract each responsibility into named functions

2. ARROW CODE
   - Nesting >3 levels deep
   - Fix: Guard clauses, early returns, strategy pattern

3. MAGIC VALUES
   - Unexplained numbers or strings in business logic
   - Fix: Named constants with descriptive names

4. MUTATION SPRAWL
   - Parameter reassignment, shared state mutation
   - Fix: Use immutable patterns, return new values

5. DEAD CODE
   - Unused imports, unreachable branches, commented code
   - Fix: Delete it. Git history preserves it if needed.

6. DUPLICATED CODE
   - Repeated >2 identical blocks >3 lines
   - Fix: Extract function or constant

7. STRINGLY TYPED
   - Using string literals where enums or constants belong
   - Fix: Use union types, enums, or constants
